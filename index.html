Tool Approval Flow
üõ°

Approval Policies

Define rules with glob patterns (salesforce.*, email.send, *) that gate tool execution behind human approval. Per-agent or domain-wide.

üìê

Field Threshold Conditions

Conditional approval based on tool input values ‚Äî e.g., "require approval when input.amount > 10000". Low-value actions proceed automatically.

‚úè

Approve with Modification

Approvers can modify tool parameters before execution ‚Äî change the amount, fix the account ID, update the date. The modified input is what executes.

Streaming & Stateless Resumption
‚è∏

Two-Phase Streaming Architecture

When a tool call matches a policy, the execution engine serialises full agent state ‚Äî messages, tool call, config ‚Äî to the database and cleanly closes the SSE stream.

Phase 1 ‚Äî Original Stream
event: token
event: token
event: tool_call_pending
event: stream_paused
Phase 2 ‚Äî After Approval
POST /approvals/{id}/resume/stream
event: tool_call_result
event: token
event: done

The consumer handles approval out-of-band, then calls the resume endpoint for a fresh SSE stream. Fully stateless ‚Äî no long-lived connections.

üîÑ

Rejection Handling

When an approver rejects a tool call with a reason, the agent receives the rejection as context and generates an intelligent alternative response ‚Äî no dead ends.

‚è∞

Automatic Expiry

A background job (APScheduler, every 60s) expires stale approval requests after the configured timeout. Expired requests trigger approval.expired webhooks.

Chain Human Input Nodes
üîó

HumanInput Chain Node

A dedicated chain node type that pauses execution and creates an approval request. The chain resumes from the next node after input is received.

üéõ

Three Input Types

Approval ‚Äî simple yes/no gate. Text ‚Äî free-form human input. Choice ‚Äî multiple options presented to the approver. All configurable per node.

‚ñ∂

Chain Resume Executor

Dedicated service deserialises chain state and continues execution from the next node after human input. Works with both admin UI and Public API.
